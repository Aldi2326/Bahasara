# SIBARAJA

SIBARAJA merupakan aplikasi Sistem Informasi Geografis berbasis web yang digunakan untuk pemetaan bahasa, sastra, dan aksara di Provinsi Jambi. Sistem ini dikembangkan sebagai upaya pelestarian dan dokumentasi kebudayaan daerah, serta memudahkan masyarakat dan instansi terkait dalam memperoleh informasi yang akurat dan terstruktur mengenai persebaran bahasa dan sastra di wilayah Jambi.
